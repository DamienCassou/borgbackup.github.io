Borg 1.1

Borg 1.1 is the current development branch of Borg and is in the beta phase. It is not recommended to use beta software in a production environment except for evaluation and testing.

Borg 1.0

The current stable series is Borg 1.0, with the initial 1.0.0 release on 5th March 2016. The latest release in this series is Borg 1.0.10, released on 13th February 2017. This is the series shipped by most Linux distributions.

Borg 0.xx

Borg 0.xx was the development series leading up to the Borg 1.0 release and is not supported any more. It was active from early 2015 to early 2016 including releases from 0.23.0 to 0.30.1.
